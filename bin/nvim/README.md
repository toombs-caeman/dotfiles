my self-contained nvim configuration.

uses [git subrepo](https://github.com/ingydotnet/git-subrepo) and vim8's pack to include plugins

# Try it
``` bash
git clone https://github.com/toombs-caeman/scnvim
cd scnvim
./nvim
```
